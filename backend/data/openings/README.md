# ECO opening data

The TSV files in this directory come from
[`lichess-org/chess-openings`](https://github.com/lichess-org/chess-openings).
They cover ECO volumes A through E and are distributed under the CC0 Public
Domain Dedication. `COPYING.txt` contains the upstream license text.

The snapshot was downloaded from the upstream `master` branch on 2026-06-21.

The application loads these files locally at runtime. Production does not make
network requests to classify an opening.
